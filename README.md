# Amir Hossein Moniri — Personal Portfolio

## About
A clean, responsive personal portfolio website showcasing my background, skills, projects, and résumé. Built with **HTML**, **CSS**, and **JavaScript**, hosted on **GitHub Pages**.

## Pages
- **Home** (`index.html`) — Landing page with introduction
- **About** (`about.html`) — Bio, skills, and interests
- **Projects** (`projects.html`) — Featured and other projects
- **Resume** (`resume.html`) — Education, experience, and skills

## Features
- Semantic HTML structure for accessibility
- Responsive design with CSS variables for easy theming
- Active navigation highlighting (shows current page)
- Smooth scrolling for better user experience
- Consistent header and footer across all pages

## File Structure
```
.
├── index.html           # Home page
├── about.html           # About page
├── projects.html        # Projects showcase
├── resume.html          # Resume/CV
├── css/
│   └── style.css        # Main stylesheet
├── js/
│   ├── main.js          # Core functionality (nav highlighting, smooth scroll)
│   └── script.js        # Additional scripts (optional)
└── assets/
    └── images/          # Profile photos and project images
```

## Technologies
- **HTML5** — Semantic markup
- **CSS3** — Responsive design with CSS variables
- **JavaScript** — Navigation interactivity

## Usage
1. Clone the repository
2. Open `index.html` in a browser or serve via GitHub Pages
3. Navigate using the menu to explore different sections
